Cost Functions¶

cppATF allows as cost function any arbitrary C++ callable that takes as input a configuration of tuning parameters and returns a value for which operator < is defined, e.g., size_t.

Pre-Implemented Cost Functions¶

cppATF provides the following pre-implemented cost functions:

class generic::cost_function¶
cost_function(const std::string &run_command)¶
Parameters:

run_command – Run command (executed in bash).

compile_command(const std::string &compile_command)¶
Parameters:

compile_command – Compile command (executed in bash).

costfile(const std::string &costfile)¶
Parameters:

costfile – Path to costfile containing cost as string (must be convertible to cost_t).

class opencl::cost_function¶
cost_function(const opencl::kernel &kernel)¶

Initializes cost function with OpenCL kernel to tune.

platform_id(size_t platform_id)¶

Target OpenCL platform id.

device_id(size_t device_id)¶

Target OpenCL device id.

template<typename ...Ts>
inputs(Ts&&... inputs)¶

Kernel’s input arguments (specified as instances of atf::scalar<T> and atf::buffer<T>).

global_size(tp_int_expression &&gs_0, tp_int_expression &&gs_1 = 1, tp_int_expression &&gs_2 = 1)¶

Kernel’s 3-dimensional OpenCL global size as arithmetic expressions that may contain tuning parameters.

local_size(tp_int_expression &&ls_0, tp_int_expression &&ls_1 = 1, tp_int_expression &&ls_2 = 1)¶

Kernel’s 3-dimensional OpenCL local size as arithmetic expressions that may contain tuning parameters.

template<size_t index>
check_result(gold_data, comparator = equality())¶

Check result for scalar/buffer at position index against gold_data.

Parameters:

  • gold_data – either of type: i) std::vector<T> for atf::buffer<T>, or ii) T for atf::scalar<T>

  • comparator – used for comparing T values; is of type std::function<bool(T,T)>

template<size_t index>
check_result(gold_callable, comparator = equality())¶

Check result for scalar/buffer at position index against scalar/buffer computed via gold_callable.

Parameters:

  • gold_callable – computes scalar/buffer (of type T/std::vector<T>) using kernel’s input scalars/buffers (of type T/std::vector<T>)

  • comparator – used for comparing T values; is of type std::function<bool(T,T)>

warmups(size_t warmups)¶

Number of warmups for each kernel run.

evaluations(size_t evaluations)¶

Number of evaluations for each kernel run.
