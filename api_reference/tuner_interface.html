Tuner¶

class tuner¶

Represents a tuner, that can be customized via its member setter functions.

tuner()¶

Default constructor.

tuning_parameters(tps...)¶

Sets program’s tuning parameters.

tuning_parameters(tp_groups...)¶

Sets program’s tuning parameters as independent parameter groups.

search_technique(const search_technique &search_technique)¶

Sets the search technique for exploration.

silent(bool silent)¶

Silences log messages.

log_file(const std::string &log_file)¶

Sets path to logfile.

tune(cost_function &cost_function, const abort_condition &abort_condition)¶

Tunes cost_function until abort_condition is met.

Default abort condition: explore full search space.

make_step(cost_function &cost_function)¶

Make one tuning step using cost_function.

configuration get_configuration()¶

Request next configuration to evaluate.

Evaluated cost must be reported to tuner via function report_cost.

report_cost(cost_t cost)¶

Report evaluated cost of requested configuration.

Configuration had to be requested via get_configuration.

tuning_status get_tuning_status()¶

Returns the tuning status object.

class tuning_parameter¶
tuning_parameter(name, range, constraint)¶

Specifies a tuning parameter by its name, range, and constraint.

Parameters:

  • name – Tuning parameter’s name as string

  • range – Either: 1) atf::interval<T>(min, max) which is an interval of values of type T between min and max (both including); intervals may have as optional argument a step_size and function generator (for using values generator(min), ..., generator(max); currently pre-implemented: atf::pow_2); 2) { v_1, v_2, ... } which is a set of values v_1, v_2, … of same type T
